        Streaming

        This section covers how to use Weird Waves as a livestream widget, e.g. on Twitch. This information is irrelevant if you're not displaying Weird Waves visually on-stream. Either way, make sure to check out the Settings as well.

        Making a widget

        To use Weird Waves like a stream widget in software like OBS:

        1. Create a browser source.
        2. Set its URL to weirdwaves.net.
        3. Delete its default custom CSS.
        4. Resize and crop it.
        5. Right-click it and select “Interact” to use it just like you are now.

        Isolating the radio

        Use this custom CSS to add empty space above and below the radio, for easier cropping:

        #radio { margin-block: 100vh; }

        Customising the theme and font

        Set the theme to “Dark” and font to “Serif”, then add this custom CSS:

        .theme-dark { --foreground-colour: [colour]; --background-colour: [colour]; --hot-detail-colour: [colour]; --cold-detail-colour: [colour]; } .font-serif { --font-stack: [font]; }

        Replace [colour] with your own colour values. These can be any valid CSS colours: hex codes, RGB values, HTML colour names, etc. Replace [font] with the name of any font installed on your computer. If the name includes a space, wrap the whole name in quotes (e.g. "Comic Sans").

        Copyright?

        All LibriVox recordings are in the public domain. The copyright status of old radio broadcasts is murky, but virtually any person or organisation who could press a claim is long dead or gone, and almost nobody cares about the atom-thin penny shavings you'd make from streaming ancient, commercially-irrelevant radio broadcasts on Twitch.

        If you're still concerned, switch on the following toggle to exclude certain series/sources when adding random shows or the entire archive to the playlist: CASiana, The Mercury Theatre, Mindwebs, Nightfall, and Seeing Ear Theater.

        Some of these (The Mercury Theatre and Nightfall) are muted on twitch due to copyright claims. The others are possible, but unconfirmed, risks.
